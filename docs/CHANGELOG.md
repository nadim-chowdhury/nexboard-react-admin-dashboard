# Changelog

All notable changes to NexBoard will be documented in this file.

## [1.0.0] — 2026-03-25

### 🚀 Initial Release

#### Dashboards
- Analytics Dashboard with revenue charts, traffic sources, and recent orders
- AI Dashboard with API usage metrics, model performance, and generations
- eCommerce Dashboard with sales trends, top products, and category breakdown

#### AI Applications
- AI Chat — ChatGPT-style interface with conversation sidebar
- AI Image Generator — Text-to-image with style/quality controls and gallery
- AI Content Writer — Content generation with templates and tone settings

#### Management
- User Management — Searchable table with roles, plans, and action menus
- Pricing Plans — 3-tier cards with monthly/yearly billing toggle
- Invoice — Professional invoice with line items and totals

#### Authentication
- Login — Social OAuth (Google, GitHub) + email/password
- Register — User registration with terms acceptance
- Forgot Password — Email reset with animated success state

#### Settings
- Profile — User profile with cover, stats, contact info, and activity timeline
- Account Settings — Tabbed settings (General, Security, Notifications, Billing)

#### Error Pages
- 404 Not Found — Branded error page with navigation
- 500 Server Error — Error boundary with retry option

#### Design System
- Aurora Design System with unique oklch jewel-tone palette
- Dark / Light mode with system preference support
- Custom utilities: gradients, glassmorphism, glow effects, dot patterns
- Smooth scrollbar styling
- Page entrance animations

#### Technical
- Next.js 16 App Router with TypeScript
- 55+ shadcn/ui components (Vega preset)
- Tailwind CSS v4 with CSS custom properties
- Recharts for data visualization
- Framer Motion for animations
- Redux Toolkit for state management
- SEO-optimized metadata
